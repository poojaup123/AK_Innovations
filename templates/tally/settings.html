Tally Integration Guide
How to Sync Data with Tally:
  1. Export Masters: First export Chart of Accounts (Ledgers) and Stock Items to create master data in Tally
  2. Import in Tally: Use Tally's Gateway of Tally > Import Data to import the XML files
  3. Export Vouchers: Export transactions as vouchers with appropriate date ranges
  4. Verify & Post: Review imported data in Tally and post vouchers
  5. Mark as Synced: Use the sync status page to mark successfully imported records

GST Configuration
GST Features Exported:
  • GSTIN numbers for suppliers/customers
  • HSN codes for stock items
  • GST rates from item master
  • GST registration type detection
Tip:

Ensure GST rates and HSN codes are correctly configured in your item master before exporting to maintain GST compliance in Tally.

Best Practices
✓ Do:
  • Export masters before vouchers
  • Verify data in Tally before posting
  • Export in chronological order
  • Mark records as synced after successful import
  • Keep regular backups of Tally data
  • Test with small date ranges first
✗ Don't:
  • Export incomplete transactions
  • Import duplicate vouchers
  • Ignore GST validation errors
  • Skip data verification in Tally
  • Export without proper date filters
  • Forget to reconcile after import
