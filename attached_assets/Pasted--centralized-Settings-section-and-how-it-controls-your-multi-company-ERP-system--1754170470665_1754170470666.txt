## 🧩 **🔧 Centralized Settings Module – Summary**

Your `Settings` section is now the **control center** for configuring everything related to:

* ✅ Companies
* ✅ Inventory behavior
* ✅ Accounting automation
* ✅ Production rules
* ✅ Job Work preferences
* ✅ User roles & access
* ✅ Multi-company management

---

### 🏢 **1. Company Settings**

| Feature                 | Description                            |
| ----------------------- | -------------------------------------- |
| Add/Edit Company        | Add GSTIN, logo, address, contact      |
| Assign Users to Company | Restrict user access by company        |
| Enable/Disable Company  | Temporarily hide a company             |
| Set Default Company     | Default selection for new transactions |

---

### 📦 **2. Inventory Settings**

| Option                    | Description                               |
| ------------------------- | ----------------------------------------- |
| ✅ Shared Inventory        | Toggle inventory as global or per-company |
| 📦 Stock Valuation Method | FIFO / LIFO / Moving Average              |
| 📏 Enable Multi-UOM       | Define unit conversions (kg → pcs, etc.)  |
| 🧾 Enable Batch Tracking  | Tracks raw material batch-wise            |
| ❗ Warn on Negative Stock  | Enable alert when issuing more than stock |

---

### 📘 **3. Accounting Settings**

| Setting                         | Description                             |
| ------------------------------- | --------------------------------------- |
| 🧾 Auto Journal Entries         | Automatically post entries (GRN, Sales) |
| 🧮 Default Ledgers (Sales, WIP) | Map accounts to transactions            |
| 📊 Financial Year Range         | Set opening/closing period              |
| 💳 GST Defaults                 | Default tax accounts & rates            |

---

### ⚙️ **4. Production Settings**

| Setting                  | Description                           |
| ------------------------ | ------------------------------------- |
| ✅ Enable Nested BOM      | Allows multi-level BOM structure      |
| 🧠 Auto Cost Calculation | Uses BOM to derive production cost    |
| 🏷️ Link Output to Batch | Track final products batch-wise       |
| 🔒 Lock Consumption      | Prevent overconsumption in production |

---

### 🛠️ **5. Job Work Settings**

| Setting                   | Description                      |
| ------------------------- | -------------------------------- |
| 📦 GRN Required on Return | Only finalize on return of goods |
| 🧾 Track Vendor Rate      | Maintain per-process rate        |
| ♻️ Scrap Entry Enabled    | Allow entry of scrap in GRN      |
| 🔁 Job Work Billing Mode  | Manual or auto invoice           |

---

### 👤 **6. User & Role Settings**

| Feature                     | Description                              |
| --------------------------- | ---------------------------------------- |
| Assign Users to Companies   | Enforce company-based access             |
| Role Permissions per Module | Control visibility (PO, Inventory, etc.) |
| Allow Company Switching     | Dropdown to change active company        |

---
