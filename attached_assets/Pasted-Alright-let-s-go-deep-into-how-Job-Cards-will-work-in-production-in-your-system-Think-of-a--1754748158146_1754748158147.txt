### **2. Contents of a Job Card**

A Job Card has:

* **Header**

  * Job Card ID
  * Production Order ID
  * Component name & code
  * Department / Vendor
  * Planned start & end date
  * Target qty
  * Batch numbers (input/output)
* **Routing Table**

  * Process sequence
  * Machine/worker assignments
  * Time estimates
* **Material Issue**

  * RM list with batch numbers
  * Qty issued
* **Daily Progress Logs**

  * Qty done
  * Scrap/rework qty
  * Operator name
* **Cost Tracking**

  * Labor time cost
  * Machine cost
  * Material cost
* **Status**

  * Pending / In Progress / Completed
